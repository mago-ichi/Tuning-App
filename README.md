# TuningApp

iOS向けのチューニングアプリです。マイクから音程を検出し、ノート名・周波数・ずれ(¢)を表示します。クロマチックとギター(EADGBE)に対応します。

## 使い方

1. Xcode で `TuningApp/TuningApp.xcodeproj` を開く
2. iPhone 実機に接続してビルド (マイク入力のため実機必須)
3. 初回起動時にマイク権限を許可
4. 大きい開始ボタンで検出を開始
5. 上部のセグメントで `クロマチック` / `ギター(EADGBE)` を切り替え

## チューニング機能

- **アーク針メーター**: ±50¢ をマップした半円メーターで、針が中央に近づくとグリーンになりロックします
- **大型ノート表示**: 検出ノートとオクターブ、Hz を一目で確認
- **自動弦判定 (ギターモード)**: 押した弦に合わせて 6 弦のターゲットを自動選択
- **手動選択**: 6 弦のカプセルから直接タップで選択
- **基準音再生**: お手本ボタンで選択弦の基準音を再生
- **基準ピッチ調整**: 設定シートから A4=415〜466Hz とカポ位置を変更
- **触覚フィードバック**: ロックインで成功ハプティクスが鳴る

## 精度のための信号処理

- 1 次 IIR ハイパスフィルタ (60Hz) でハム / 低周波雑音を除去
- YIN ピッチ推定 (閾値 0.15 / 信頼度 ≥ 0.6) で基本周波数を高精度に推定
- 倍音マッチング (×1〜×3) でギター弦の自動判定
- ¢ の指数移動平均で揺れを抑制
- 8 フレーム連続グリーンでロックイン
- 音切れ時に直前表示を 600ms ホールド

## 実装

- `ContentView.swift`: メインUI (アーク針メーター、ノートカード、弦カルーセル、設定シート)
- `PitchDetector.swift`: 音声入力・YIN推定・後段フィルタ・状態管理
- `ToneGenerator.swift`: お手本音源生成 (チューニングフォーク風)
- `Info.plist`: マイク権限説明

## 実機で確認するポイント

- アーク針メーターが揺らがず動く: 信号処理が効いています
- 弦を弾くとカプセルがハイライト: 自動判定が機能しています
- 中央が緑になる: ¢=0 付近を捉えています
- 設定の入力レベルバーで音量を確認できます
