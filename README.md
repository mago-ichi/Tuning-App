# TuningApp

iOS向けのシンプルなチューニングアプリです。マイク入力から音程を推定し、ノート名とHz、ピッチのずれ（cent）を表示します。

## 使い方

1. Xcodeで `TuningApp/TuningApp.xcodeproj` を開く
2. iPhone実機を接続して実行（マイク利用のため実機必須）
3. 初回起動時にマイク権限を許可
4. `開始` ボタンで検出を開始
5. `モード` で `クロマチック` / `ギター(EADGBE)` を切り替え

## 実装概要

- `ContentView.swift`: UI表示と操作ボタン
- `PitchDetector.swift`: AVAudioEngineで音声取得、YIN方式の周波数推定、入力レベル/信頼度表示
- `Info.plist`: マイク権限説明を設定

## 実機での確認ポイント

- 入力レベルバーが動く: マイク入力は取れています
- `マイク感度` は `0.1x` から `8.0x` まで調整できます
- `信頼度` が上がる: 音程解析に使える単音として認識できています
- ギターモードでは `A4` の基準Hz、カポ位置、弦ごとの目標Hzを画面で確認できます
- ギターモードでは倍音も考慮して、E2/A2/D3/G3/B3/E4 の最も近い弦を目標にします
